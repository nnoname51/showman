# 🌿 Plants Bot

**Plants** — это умный Telegram-бот на Python, созданный для тех, кто любит свои домашние растения, но иногда забывает о поливе. Бот помогает вести список ваших цветов, хранит историю ухода и знает секреты выращивания 15 самых популярных видов растений.

## ✨ Основные фишки

*   **Умное управление без команд**: Больше не нужно учить сложные команды со слешем (`/`). Просто нажимай на кнопки в меню.
*   **Цифровой сад**: Добавляй свои растения в личную базу данных.
*   **Трекер полива**: Отмечай полив в один клик. Бот запомнит точную дату и время.
*   **База знаний**: Встроенные советы по уходу за 15 видами (орхидеи, кактусы, розы и др.).
*   **Железная память**: Бот сохраняет все данные в файл `plants_data.json`. Даже если выключить компьютер, информация не пропадет.

## 🛠 Технологии

*   **Язык**: Python 3.10+
*   **Библиотека**: `python-telegram-bot` (версия 20+)
*   **Хранение данных**: JSON (локальная база данных)
